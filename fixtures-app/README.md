# SA Sports Fixtures

Your personal sports fixture tracker, tailored for South Africa.

---

## 🚀 Deploy to Vercel (free, ~3 minutes)

### Step 1 — Upload to GitHub
1. Go to [github.com](https://github.com) and sign up / log in (free)
2. Click **New repository** → name it `sa-sports-fixtures` → **Create repository**
3. On the next screen, click **uploading an existing file**
4. Drag and drop **all the files from this folder** into the upload area
5. Click **Commit changes**

### Step 2 — Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign up with your GitHub account (free)
2. Click **Add New Project**
3. Select your `sa-sports-fixtures` repository
4. Leave all settings as default — Vercel detects React automatically
5. Click **Deploy**
6. In ~60 seconds you'll have a live URL like `sa-sports-fixtures.vercel.app` ✅

## 🔑 Get your Anthropic API Key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Sign up / log in
3. Click **API Keys** → **Create Key**
4. Copy the key (starts with `sk-ant-api03-...`)
5. Open your deployed app → click **⚠ API Key** → paste it in → **Save**

New accounts get **$5 free credit** — easily enough for months of daily fixture lookups.

## Features

- **Today / Weekend / This Week** views
- **Sport filter pills** — Rugby, Cricket, Football, F1, Golf, Tennis, Athletics
- **🟢 Springbok fixtures** highlighted in green
- **⭐ Grey College** fixtures flagged in gold
- **SA team** fixtures (Bulls, Lions, Sharks, Stormers, Cheetahs) highlighted
- Expandable fixture cards with venue, competition, and result
- All times in **SAST (UTC+2)**
- Powered by TheSportsDB (free) + Claude AI with web search
